📦 Package Delivery Optimization System:

This project is a Python-based system that optimizes package delivery by assigning packages to vehicles and finding efficient delivery routes. It solves a version of the Capacitated Vehicle Routing Problem (CVRP) while considering vehicle capacity and package priority.


Designed an optimization system to assign packages to vehicles and minimize total travel distance.
Implemented and compared Simulated Annealing and Genetic Algorithm to improve routing solutions under real-world constraints.

🛠️ Technologies & Tools:

Python
Simulated Annealing & Genetic Algorithm (metaheuristic optimization)
Matplotlib & Tkinter (for visualization)
